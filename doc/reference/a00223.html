asio::ipv4::multicast

asio::ipv4::multicast Namespace Reference

Detailed Description

The asio::ipv4::multicast namespace defines the socket options used for multicast.


Namespaces

namespace  socket_option
 The asio::ipv4::multicast::socket_option namespace defines socket options helper templates.
