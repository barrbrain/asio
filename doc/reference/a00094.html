Detailed Description

template<typename Demuxer_Service>
class asio::basic_demuxer< Demuxer_Service >::work

Class to inform the demuxer when it has work to do.

The work class is used to inform the demuxer when work starts and finishes. This ensures that the demuxer's run() function will not exit while work is underway, and that it does exit when there is no unfinished work remaining.

The work class is copy-constructible so that it may be used as a data member in a handler class. It is not assignable.
