Detailed Description

Describes an endpoint for a UDP socket.

The asio::ipv4::udp::endpoint class describes an endpoint that may be associated with a particular socket.

Thread Safety:
Distinct objects: Safe.
Shared objects: Unsafe.
Concepts:
