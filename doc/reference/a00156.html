asio::mutable_buffer

asio::mutable_buffer Class Reference

List of all members.

Detailed Description

Holds a buffer that can be modified.

The const_buffer class provides a safe representation of a buffer that can be modified. It does not own the underlying data, and so is cheap to copy or assign.
