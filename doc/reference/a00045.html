asio::socket_base Class Reference

Inherited by asio::basic_socket< Protocol, SocketService >, asio::basic_socket< Protocol, DatagramSocketService >, asio::basic_socket< Protocol, StreamSocketService >, and asio::basic_socket_acceptor< Protocol, SocketAcceptorService >.

Detailed Description

The socket_base class is used as a base for the basic_stream_socket and basic_datagram_socket class templates so that we have a common place to define the shutdown_type and enum.
