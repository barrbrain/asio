Detailed Description

Socket_Option concept.

Defines the interface that must be implemented by an object passed as the option parameter to:

  • asio::tcp::socket::get_option
  • asio::tcp::socket::set_option
  • asio::udp::socket::get_option
  • asio::udp::socket::set_option
  • asio::tcp::acceptor::get_option
  • asio::tcp::acceptor::set_option
