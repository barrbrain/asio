Detailed Description

template<std::size_t N>
class asio::mutable_buffers< N >

Holds a list of buffers that can be modified.

The mutable_buffers class template provides a list of buffers that can be passed to operations expecting an implementation of the Mutable_Buffers concept. It does not own the underlying data, and so is cheap to copy or assign.
