asio Reference Namespace List

Here is a list of all documented namespaces with brief descriptions:
asioThe asio namespace defines all user-accessible classes and templates
asio::ipThe asio::ip namespace defines the classes associated with IP
asio::ip::multicastThe asio::ip::multicast namespace defines the socket options used for multicast
asio::ip::unicastThe asio::ip::unicast namespace defines the socket options used for unicast
asio::placeholdersThe asio::placeholders namespace defines argument placeholders for use with boost::bind
asio::sslThe asio::ssl namespace defines templates and classes for SSL
