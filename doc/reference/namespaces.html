asio::ipv4::socket_optionThe asio::ipv4::socket_option namespace defines socket options helper templates
asio::placeholdersThe asio::placeholders namespace defines argument placeholders for use with boost::bind
asio::socket_optionThe asio::socket_option namespace defines socket options helper templates
asio::sslThe asio::ssl namespace defines templates and classes for SSL
