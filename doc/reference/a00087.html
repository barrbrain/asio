Detailed Description

Asynchronous object concept.

The asynchronous object concept provides callers with access to the demuxer used to dispatch handlers for asynchronous operations. This allows the asynchronous object's operations to be wrapped in a higher-level operation, such that the handler for the higher-level operation be dispatched through the correct demuxer.

Get the demuxer associated with the asynchronous object.

This function may be used to obtain the demuxer object that the object uses to dispatch handlers for asynchronous operations.

Returns:
A reference to the demuxer object that the object will use to dispatch handlers. Ownership is not transferred to the caller.
