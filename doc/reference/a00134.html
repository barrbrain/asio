Endpoint Class Reference

Detailed Description

Endpoint concept.

Defines the interface that must be implemented by an object passed as a parameter to:

  • asio::stream_socket::bind
  • asio::stream_socket::connect
  • asio::stream_socket::get_local_endpoint
  • asio::stream_socket::get_remote_endpoint
  • asio::datagram_socket::bind
  • asio::datagram_socket::connect
  • asio::datagram_socket::get_local_endpoint
  • asio::datagram_socket::get_remote_endpoint
  • asio::socket_acceptor::bind
  • asio::socket_acceptor::get_local_endpoint
  • asio::socket_acceptor::accept_address
Implemented By:
asio::ipv4::tcp::endpoint
asio::ipv4::udp::endpoint
