Daytime.1 - A synchronous TCP daytime client

This tutorial program shows how to use asio to implement a client application with TCP.

We start by including the necessary header files.

#include <iostream>
#include "asio.hpp"

The purpose of this application is to access a daytime service, so we need the user to specify the server.

int main(int argc, char* argv[])
{
  try
  {
    if (argc != 2)
    {
      std::cerr << "Usage: client <host>" << std::endl;
      return 1;
    }

All programs that use asio need to have at least one asio::demuxer object.

    asio::demuxer demuxer;

We need to turn the server name that was specified as a parameter to the application, into an IP address. To do this we use an asio::ipv4::host_resolver object.

    asio::ipv4::host_resolver host_resolver(demuxer);

Now we need an object of type asio::ipv4::host for the server. The asio::ipv4::host_resolver::get_host_by_name() function takes a hostname and populates the host object with the corresponding IP address.

    asio::ipv4::host host;
    host_resolver.get_host_by_name(host, argv[1]);

The socket is to connect to the daytime service, TCP port 13, on the server. Create an object for this remote endpoint.

    asio::ipv4::tcp::endpoint remote_endpoint(13, host.address(0));

Now create and connect the socket.

    asio::stream_socket socket(demuxer);
    socket.connect(remote_endpoint);

The connection is open. All we need to do now is read the response from the daytime service. When the server closes the connection, the asio::stream_socket::read_some() function will exit with the asio::error::eof error, which is how we know to exit the loop.

    for (;;)
    {
      char buf[128];
      asio::error error;
      size_t len = socket.read_some(
          asio::buffer(buf), asio::assign_error(error));
      if (error == asio::error::eof)
        break; // Connection closed cleanly by peer.
      else if (error)
        throw error; // Some other error.
      std::cout.write(buf, len);
    }

Finally, handle any exceptions that may have been thrown.

  }
  catch (asio::error& e)
  {
    std::cerr << e << std::endl;
  }
