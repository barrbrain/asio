Daytime.3 - An asynchronous TCP daytime server

The main() function

int main()
{
  try
  {
    asio::demuxer demuxer;

    asio::socket_acceptor acceptor(demuxer, asio::ipv4::tcp::endpoint(13));

We need to set up the initial socket to hold an accepted client connection.

    asio::stream_socket* socket = new asio::stream_socket(demuxer);

The function asio::socket_acceptor::async_accept() will cause the application to listen in the background for a new connection. When such a connection is received, the demuxer will invoke the handle_accept() function with an asio::error argument.

    acceptor.async_accept(*socket,
        boost::bind(handle_accept, &acceptor, socket,
          asio::placeholders::error));

Now that we have created some work for it to do, we can run the demuxer.

    demuxer.run();
  }
  catch (asio::error& e)
  {
    std::cerr << e << std::endl;
  }

  return 0;
}

The handle_accept() function

The function handle_accept() will service the client request.

void handle_accept(asio::socket_acceptor* acceptor,
    asio::stream_socket* socket, const asio::error& error)
{

The error parameter contains the result of the asynchronous operation.

  if (!error)
  {

Determine what we are going to send.

    using namespace std; // For time_t, time, ctime, strdup and strlen.
    time_t now = time(0);
    char* write_buf = strdup(ctime(&now));
    size_t write_length = strlen(write_buf);

We now call asio::async_write_n to serve the data to the client. Note that we are using asio::async_write_n, rather than asio::async_write, to ensure that the entire block of data is sent.

    asio::async_write_n(*socket, asio::buffers(write_buf, write_length),
        boost::bind(handle_write, socket, write_buf,
          asio::placeholders::error,
          asio::placeholders::last_bytes_transferred,
          asio::placeholders::total_bytes_transferred));

When initiating the asynchronous operation, and if using boost::bind, you must specify only the arguments that match the handler's parameter list. In this program, all three of the argument placeholders (asio::placeholders::error, asio::placeholders::last_bytes_transferred and asio::placeholders::total_bytes_transferred) could potentially have been removed.

Any further actions for this client connection are now the responsibility of handle_write().

Create a new socket for the next client connection.

    socket = new asio::stream_socket(acceptor->demuxer());

    acceptor->async_accept(*socket,
        boost::bind(handle_accept, acceptor, socket,
          asio::placeholders::error));
  }
  else
  {
    delete socket;
  }
}

The handle_write() function

The function handle_write() is invoked after the service request has been completed.

void handle_write(asio::stream_socket* socket, char* write_buf,
    const asio::error& /*error*/, size_t /*last_bytes_transferred*/,
    size_t /*total_bytes_transferred*/)
{
  using namespace std; // For free.
  free(write_buf);
  delete socket;
}
