asio::arg Namespace Reference

Detailed Description

The asio::arg namespace defines argument placeholders for use with boost::bind.


Variables

boost::arg< 1 > error
 An argument placeholder, for use with boost::bind, that corresponds to the error argument of a handler for any of the asynchronous socket-related functions.

boost::arg< 2 > bytes_sent
 An argument placeholder, for use with boost::bind, that corresponds to the bytes_sent argument of a handler for asynchronous functions such as asio::async_send or asio::stream_socket::async_send.
