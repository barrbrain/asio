asio::placeholders Namespace Reference

Detailed Description

The asio::placeholders namespace defines argument placeholders for use with boost::bind.


Variables

boost::arg< 1 > error
 An argument placeholder, for use with boost::bind, that corresponds to the error argument of a handler for any of the asynchronous functions.

boost::arg< 2 > bytes_transferred
 An argument placeholder, for use with boost::bind, that corresponds to the bytes_transferred argument of a handler for asynchronous functions such as asio::async_write or asio::stream_socket::async_write.

boost::arg< 2 > last_bytes_transferred
 An argument placeholder, for use with boost::bind, that corresponds to the last_bytes_transferred argument of a handler for asynchronous functions such as asio::async_write_n or asio::async_read_at_least_n.

boost::arg< 3 > total_bytes_transferred
 An argument placeholder, for use with boost::bind, that corresponds to the total_bytes_transferred argument of a handler for asynchronous functions such as asio::async_write_n or asio::async_read_at_least_n.
