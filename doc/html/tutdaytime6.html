Tutorial Daytime.6 - An asynchronous UDP daytime server

Removing Unused Handler Parameters

In this tutorial program, the error and bytes_sent parameters are not used in the body of the handle_sendto() function. As those parameters are not needed, they can be removed them from the function:

void handle_sendto(char* send_buf)
{
  using namespace std; // For free.
  free(send_buf);
}

When initiating the asynchronous operation, and if using boost::bind, you must specify only the arguments that match the handler's parameter list. In this tutorial program, both of the argument placeholders (asio::placeholders::error and asio::placeholders::bytes_transferred) should therefore be removed:

socket->async_sendto(send_buf, send_length, *remote_address,
    boost::bind(handle_sendto, send_buf));

The handle_recvfrom() functions's bytes_transferred parameter can be similarly removed.
