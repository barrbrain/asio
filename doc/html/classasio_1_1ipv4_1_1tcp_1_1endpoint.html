asio::ipv4::tcp::endpoint Class Reference

List of all members.

Detailed Description

Describes an endpoint for a TCP socket.

The asio::ipv4::tcp::endpoint class describes an endpoint that may be associated with a particular socket.

Thread Safety:
Distinct objects: Safe.
Shared objects: Unsafe.
Concepts:
