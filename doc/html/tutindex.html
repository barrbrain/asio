Basic Skills

The tutorials in this first section introduce the fundamental concepts required to use the asio toolkit. Before plunging into the complex world of network programming, these tutorial programs illustrate the basic skills using simple asynchronous timers.

  • Tutorial Timer.1 - Using a timer synchronously
  • Tutorial Timer.2 - Using a timer asynchronously
  • Tutorial Timer.3 - Binding arguments to a handler
  • Tutorial Timer.4 - Using a member function as a handler
  • Tutorial Timer.5 - Synchronising handlers in multithreaded programs

Introduction to Sockets

The tutorials in this section show how to use asio to develop simple client and server programs. These tutorial programs are based around the daytime protocol, which supports both TCP and UDP.

The first three tutorials implement the daytime protocol using TCP.

  • Tutorial Daytime.1 - A synchronous TCP daytime client
  • Tutorial Daytime.2 - A synchronous TCP daytime server
  • Tutorial Daytime.3 - An asynchronous TCP daytime server
The next three tutorials implement the daytime protocol using UDP.

  • Tutorial Daytime.4 - A synchronous UDP daytime client
  • Tutorial Daytime.5 - A synchronous UDP daytime server
  • Tutorial Daytime.6 - An asynchronous UDP daytime server
The last tutorial in this section demonstrates how asio allows the TCP and UDP servers to be easily combined into a single program.

  • Tutorial Daytime.7 - A combined TCP/UDP asynchronous server
