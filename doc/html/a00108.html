Detailed Description

template<typename Stream, typename Buffer>
class asio::buffered_stream< Stream, Buffer >

Adds buffering to the read- and write-related operations of a stream.

The buffered_stream class template can be used to add buffering to the synchronous and asynchronous read and write operations of a stream.

Thread Safety:
Distinct objects: Safe.
Shared objects: Unsafe.
Concepts:
Async_Object, Async_Read_Stream, Async_Write_Stream, Stream, Sync_Read_Stream, Sync_Write_Stream.
