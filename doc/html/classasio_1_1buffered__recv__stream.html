Detailed Description

template<typename Stream, typename Buffer>
class asio::buffered_recv_stream< Stream, Buffer >

Adds buffering to the receive-related operations of a stream.

The buffered_recv_stream class template can be used to add buffering to the synchronous and asynchronous receive operations of a stream.

Thread Safety:
Distinct objects: Safe.
Shared objects: Unsafe.
Concepts:
Async_Object, Async_Recv_Stream, Async_Send_Stream, Stream, Sync_Recv_Stream, Sync_Send_Stream.
