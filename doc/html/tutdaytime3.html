Tutorial Daytime.3 - An asynchronous TCP daytime server

Removing Unused Handler Parameters

You may have noticed that the error, last_bytes_sent and total_bytes_sent parameters are not used in the body of the handle_send() function. If parameters are not needed, it is possible to remove them from the function:

void handle_send(asio::stream_socket* socket, char* send_buf)
{
  using namespace std; // For free.
  free(send_buf);
  delete socket;
}

When initiating the asynchronous operation, and if using boost::bind, you must specify only the arguments that match the handler's parameter list. In this program, all three of the argument placeholders (asio::placeholders::error, asio::placeholders::last_bytes_transferred and asio::placeholders::total_bytes_transferred) should therefore be removed:

asio::async_send_n(*socket, send_buf, send_length,
    boost::bind(handle_send, socket, send_buf));
