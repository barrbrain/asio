Send the given data on the stream.

This function is used to send data on the stream. The function call will block until some or all of the data has been sent successfully, or until an error occurs.

Parameters:
data The data to be sent.
length The size of the data to be sent, in bytes.
Returns:
The number of bytes sent or 0 if the connection was closed cleanly.
Exceptions:
implementation_specified Thrown on failure.
