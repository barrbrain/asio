) 
 

Start an asynchronous send.

This function is used to asynchronously send data on the stream. The function call always returns immediately.

Parameters:
data The data to be sent on the stream. Ownership of the data is retained by the caller, which must guarantee that it is valid until the handler is called.
length The size of the data to be sent, in bytes.
handler The handler to be called when the send operation completes. Copies will be made of the handler as required. The equivalent function signature of the handler must be: 
 void handler(
   const implementation_defined& error, // Result of operation
   size_t bytes_sent                    // Number of bytes sent
 );
