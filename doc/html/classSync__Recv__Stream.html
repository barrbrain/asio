) 
 

Receive some data from the stream.

This function is used to receive data from the stream. The function call will block until data has been received successfully or an error occurs.

Parameters:
data The buffer into which the received data will be written.
max_length The maximum size of the data to be received, in bytes.
Returns:
The number of bytes received or 0 if the connection was closed cleanly.
Exceptions:
implementation_defined Thrown on failure.
